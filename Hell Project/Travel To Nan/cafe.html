เฮือนฮังต่อ กาแฟน่าน

เฮือนฮังต่อ กาแฟน่าน ร้านกาแฟ ที่คอกาแฟต้องห้ามพลาด ตั้งอยู่ในตัวเมือง เป็น ร้านกาแฟที่ชิลมากๆ ตกแต่งในสไตล์บ้านโบราณเก๋ๆ และของโบราณ ของสะสมเก่าๆ มากมาย ทางร้านมีมุมนั่งแบบห้องแอร์ (Indoor) และแบบสวน (Outdoor) ร้านใหญ่มากบรรยากาศ ภายใต้แมกไม้อันร่มรื่น ธรรมชาติสุดๆ เมนูกาแฟคือดีแบบไม่ต้องพูดเยอะ


  • พิกัด : https://goo.gl/maps/ZAqeGoaowX1s2yxZ9
  • ที่อยู่ : ตำบล ในเวียง อำเภอเมืองน่าน จังหวัดน่าน
  • ร้านเปิดบริการ : 08.00 - 18.00 น.
  • โทร : 09-0330-0887
  • ที่จอดรถ : มี
  • เว็บไซต์ : https://www.facebook.com/huan.hung.tor

ร้านของหวานป้านิ่ม

ร้านของหวานป้านิ่ม เป็นร้านขนมหวาน ร้านเด็ดเจ้าดังที่ใครไปเที่ยวน่านแล้วต้อง ห้ามพลาด อร่อยกับเมนูเด็ดอย่างบัวลอยไข่หวานที่หวานหอมหนุบหนับ หรือจะเป็นไอศครีม บัวลอย ไอศครีมข้าวเหนียวดำ ไอศครีมเต้าส่วน และอีกมากมาย ให้คนรักขนมหวานได้เลือก ทานอย่างจุใจ และรับรองว่าฟินแน่นอนจ้าา


  • พิกัด : https://goo.gl/maps/Amg4zzFZMSYVUKSn8
  • ที่อยู่ : หมู่บ้านมงคลนิมิตร ตำบลผาสิงห์ อำเภอเมือง จังหวัดน่าน
  • ร้านเปิดบริการ : 11.00 - 22.00 น. (ปิดทุกวันพุธ)
  • โทร : 08-5036-6108
  • ที่จอดรถ : มี
  • เว็บไซต์ : https://www.facebook.com/ร้านของหวานป้านิ่ม

กาแฟ บ้านไทลื้อ

กาแฟ บ้านไทลื้อ เป็นร้านกาแฟโฮมเมดของร้านลำดวลผ้าทอ แน่นอนว่า มันมิใช่เป็น แค่ร้านกาแฟธรรมดาๆ แน่นอน เพราะที่นี่ ยังมีของที่ระลึกและผ้าทอไทลื้อ ผ้าทอน้ำไหล ลายโบราณ ของดี อ.ปัว ขายอยู่ด้วย ซึ่งผ้าทอและสินค้าทั้งหมดก็มาจากผลิตภัณฑ์ชุมชน


  • พิกัด : https://maps.app.goo.gl/96p6TuhyxApDFER9A
  • ที่อยู่ : 164 หมู่ 4 ตำบลศิลาแลง อำเภอปัว จังหวัดน่าน 55120
  • ร้านเปิดบริการ : 09.00 – 18.00 น.
  • โทร : 081-8827767 / 089-2646058
  • ที่จอดรถ : มี
  • เว็บไซต์ : www.facebook.com/กาแฟบ้านไทลื้อ - ban tai lue cafe’

กลิ่นไอเกลือ

กลิ่นไอเกลือ เปิดมานานถึง 3 ปี เป็นร้านที่อยู่ในย่านบ่อเกลือใต้ ฝั่งตรงข้ามเยื้องๆ กับโรงต้มเกลือสินเธาว์ บรรยากาศของร้านดูโล่งสบาย ซึ่งเป็นชั้นล่างของอาคารสองชั้น ขนาดกว้างขวาง สามารถเปิดพัดลมได้ มีอาหารหลากหลายเมนู เบเกอรี่ ขนมปังรสต่างๆ เมนูเครื่องดื่มร้อนเย็น และกาแฟสด ไว้คอยให้บริการนักท่องเที่ยวทุกวัน


  • พิกัด : https://maps.app.goo.gl/tX5NPv8ZDMazADUc6
  • ที่อยู่ : เลขที่ 42 หมู่ที่ 1 ต.บ่อเกลือใต้ อ.บ่อเกลือ จ.น่าน
  • ร้านเปิดบริการ : 08.00 – 19.00 น.
  • โทร : 086-556-3966
  • ที่จอดรถ : มี
  • เว็บไซต์ : https://www.facebook.com/กลิ่นไอเกลือ
