เฮือนเจ้านาง

เฮือนเจ้านาง ร้านอาหารริมแม่น้ำน่าน ตั้งอยู่เส้นทางเดียวกับที่จะไปวัดพระธาตุแช่ แห้งบริการทั้งอาหารไทยและอาหารพื้นเมือง เมนูเด็ดที่ต้องสั่ง เช่น ลาบหมูคั่ว ไก่ทอด มะแขว่นแกงแคต้มยำปลาคังน้ำข้น และปลาสามรส อิ่มอร่อยกันในบรรยากาศสบายๆ พร้อม ชมวิวสวยๆของแม่น้ำน่าน


  • พิกัด : https://goo.gl/maps/cusHbD3K8uYByUko8
  • ที่อยู่ : ริมแม่น้ำน่าน (ฝั่งท่าล้อ) อำเภอภูเพียง จังหวัดน่าน
  • ร้านเปิดบริการ : 11.00 - 23.00 น.
  • โทร : 08-1808-5588
  • ที่จอดรถ : มี
  • เว็บไซต์ : https://www.facebook.com/hueanchaonangnan/

เฮือนเจ้านาง

เฮือนฮอม ร้านอาหารพื้นเมืองชื่อดังที่ไม่ควรพลาด เพราะจะได้ลิ้มลองกับอาหาร เหนือแท้ๆ หลากเมนู ไม่ว่าจะเป็น น้ำพริกหนุ่ม น้ำพริกอ่อง ข้าวซอย ขนมจีนน้ำเงี้ยว รวมถึงไก่ทอดมะแขว่นของดีเมืองน่าน อิ่มอร่อยกันในบรรยากาศสบายๆ สไตล์เมืองเหนือ


  • พิกัด : https://goo.gl/maps/fYaTumcw44MaiBsz7
  • ที่อยู่ : ถนน สุริยพงษ์ ตำบลผาสิงห์ อำเภอเมือง จังหวัดน่าน
  • ร้านเปิดบริการ : 10.00 - 15.00 น. / 18.00 - 21.00 น.
  • โทร : 0-5475-1122
  • ที่จอดรถ : มี
  • เว็บไซต์ : https://www.facebook.com/เฮือนฮอม

สุริยาการ์เด้น

สุริยาการ์เด้น สวนอาหารริมน้ำน่าน บรรยากาศดี ร้านกว้างขวาง เหมาะกับการ ไปนั่งรับประทานอาหารชิลล์ๆ ที่นี่มีอาหารหลากหลายเมนู ทั้งไทยและพื้นเมือง เช่น ลาบหมูคั่ว แกงส้มเมืองปลาคัง ปลาทอดน้ำปลา และอีกมากมาย ยิ่งช่วงเย็นจะมีดนตรี สดเล่นให้ฟังเพลินๆ อีกด้วย


  • พิกัด : https://goo.gl/maps/fYaTumcw44MaiBsz7
  • ที่อยู่ : ถนนสุมนเทวราช ตำบลในเวียง อำเภอเมือง จังหวัดน่าน
  • ร้านเปิดบริการ : 11.00 - 23.00 น.
  • โทร : 0-5471-0687 / 0-5477-2980
  • ที่จอดรถ : มี
  • เว็บไซต์ : https://www.facebook.com/สุริยาการ์เด้น จ.น่าน

บ้านๆ น่านๆ

บ้านๆ น่านๆ เป็นเหมือนสถานที่ที่รวมเอา ร้านกาแฟ ห้องสมุด และ เกสต์โฮม เข้ามาอยู่ด้วยกันอย่างลงตัว อาคารบ้านไม้ 2 ชั้น ที่มีบรรยากาศเงียบสงบแบบห้องสมุด แวดล้อมด้วยต้นไม้ร่มรื่น พร้อมกลิ่นหอมของกาแฟที่ลอยมาแตะจมูกอยู่เป็นระยะ ชวนให้ ต้องสั่งกาแฟสักแก้ว หยิบหนังสือสักเล่ม มาหามุมสงบ นั่งจิบกาแฟไป อ่านหนังสือไป


  • พิกัด : https://maps.app.goo.gl/gfDaXdknKWk5TjV57
  • ที่อยู่ : ตำบล ในเวียง อำเภอเมืองน่าน น่าน 55000
  • ร้านเปิดบริการ : 08.00 น. – 17.00 น.
  • โทร : 089-859-5898
  • ที่จอดรถ : มี
